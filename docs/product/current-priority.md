# SafeHire — Current Priority

**Last Updated:** 2026-03-22
**Sprint Focus:** Web product hardening — make SafeHire feel like a shippable, premium, India-first hiring marketplace

---

## Priority Stack (ordered)

1. **Search/discovery is the product centerpiece** — category-first, locality-first
2. **Worker cards must feel practical** — not social profiles, not LinkedIn
3. **Access request UX must not be stupid** — structured intent, not freeform paragraphs
4. **Agency experience must be first-class** — not hidden or second-class
5. **Worker dignity in every surface** — language, tone, and flow must respect workers
6. **Verification and ratings are separate** — never conflate them
7. **Demo/UAT flow must be obvious** — a tester understands value in 2 minutes
8. **Typography, spacing, and hierarchy** — premium, not prototype
9. **Prepare clean integration points** — maps, payments, KYC adapters
10. **No drift** — no LinkedIn, no profile vanity, no feature sprawl

## Current Actor Model

| Actor | Entry Point | Core Action |
|-------|------------|-------------|
| Worker | Create trust card | Get verified, get found, get hired |
| Hirer / Customer | Search by category + area | Find trusted help, check trust cards |
| Agency | Manage worker roster | Build agency brand, verify workers, serve customers |

## Current Trust Model

- **Verification** = 10-step ladder (phone, selfie, ID, face match, address, emergency contact, category, reference, agency review, re-verification)
- **Ratings** = 1-5 stars per service interaction (separate from verification)
- **References** = endorsements from past employers
- **Incidents** = factual, reviewed, appealable — never a blacklist

## What "Done" Looks Like This Sprint

- Homepage answers "what, who, how" in 5 seconds
- Search is the centerpiece with category pills, locality filter, practical cards
- Worker detail shows trust summary, verification ladder, rating, and availability
- Access request is structured (intent options, not freeform)
- Agency has a real dashboard and public presence
- Worker dashboard guides toward verification completion
- Hirer dashboard guides toward search and hiring
- Demo flow is obvious and satisfying
- Typography and spacing are premium and consistent
